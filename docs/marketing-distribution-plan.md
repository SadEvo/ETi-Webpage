# Elite Ti — Marketing & Distribution Plan

For the marketing agent. Answers: **where do we post, how often, with what, and what tools /
access do we need** to make the event calendar actually drive sales.

Current state: everything runs through **Metricool**. Metricool is great for scheduling and
analytics on IG/FB/TikTok and a link-in-bio — but it **cannot post into Facebook Groups or
forums**, which is exactly where our highest-intent buyers live. This plan keeps Metricool as
the hub and fills the gaps around it.

## What the data tells us about distribution

- **Social drives $117K / 99 orders ($1,022 AOV)** — our #1 identifiable channel. Lean in.
- **Search drives $39K / 54 orders** — a real, under-developed second channel (SEO + maybe paid).
- A large chunk of sales show as **direct/unattributed** — almost certainly group/forum/DM
  traffic and people typing the URL after seeing us in a community. **We need UTM links** to
  finally see what's actually working (see "Attribution" below).

## 2. Content pillars + cadence

Rotate these so the feed isn't all "buy now." Target ~4–6 posts/week per primary channel.

| Pillar | What | ~Mix |
|---|---|---|
| **New drops / product hero** | Clean product shots, specs, "now available" | 25% |
| **Build features / UGC** | Customer cars running our parts, reposts, tags | 25% |
| **Group buy / event** | Countdowns, tier-unlock progress, last-call | 20% |
| **Tech / install** | How it fits, install clips, fitment confidence | 15% |
| **Behind the scenes** | Titanium/carbon manufacturing, packing, the craft | 10% |
| **Community / engagement** | Polls, memes, "which build wins," questions | 5% |

**Per-event content arc** (mirrors the group-buy 4-beat): Tease → Launch → Mid-point update →
Last 48hr → Close/fulfillment recap. Every primary event in the calendar should get this full
arc across IG + Groups + Email/SMS at minimum.

## 3. Tooling — keep Metricool, add the gaps

| Tool | Status | Purpose |
|---|---|---|
| **Metricool** | ✅ Have | Hub: schedule IG/FB/TikTok, analytics, link-in-bio. Keep it. |
| **Klaviyo** (or Shopify Email + SMS to start) | ➕ Add — top priority | Email/SMS flows: abandoned cart, post-purchase, group-buy early access, win-back. Highest ROI gap for a $1K-AOV store. |
| **Canva** (Brand Kit) | ➕ Add | Consistent creative, templated countdowns/tier graphics for fast turnarounds. |
| **Group-buy landing page** | ➕ Add | A Shopify page (or Linkpop/link-in-bio) that aggregates the active event + UTM links to share into groups. |
| **Discord** | ➕ Consider | Owned community for repeat group-buy demand. |
| **Meta Business Suite / Ads Manager** | ➕ Consider | Retargeting warm visitors + lookalikes off our buyer list. High-AOV justifies modest ad spend. |
| **A forum/group posting routine** | ➕ Process, not tool | Metricool can't post to FB Groups/forums. Maintain a group/forum list + a manual posting cadence (or a community manager who has posting access). |

> **Note on Facebook Groups:** there's no clean automation — most groups require manual,
> human posting and many gate vendor posts. The realistic fix is a **maintained list of target
> groups + a manual posting checklist per event**, owned by the agent or a community manager.

## 5. Attribution — stop flying blind

A big chunk of revenue is "direct/unattributed." Fix it so we know what's working:

- **UTM-tag every link** the agent shares. Standard format:
  `?utm_source=instagram&utm_medium=social&utm_campaign=sevens-day-2026`
  - `utm_source` = platform (instagram, facebook_group, rx7club, email, sms, tiktok)
  - `utm_medium` = social / forum / email / sms / referral
  - `utm_campaign` = the event (e.g. `sc300-doorcard-gb`, `sevens-day-2026`, `bfcm-2026`)
- Build links with a UTM builder (or a saved Canva/Sheets template) and use them everywhere —
  link-in-bio, group posts, email, forum signatures.
- This turns Shopify's generic "social $117K" into **per-platform, per-campaign** numbers, so
  we can double down on what converts (likely Groups + Email) instead of guessing.

## 6. KPIs + reporting rhythm

Report weekly (Metricool + Shopify), reviewed monthly:

| KPI | Why |
|---|---|
| Revenue by UTM source/campaign | Which channels + events actually drive sales |
| Email/SMS revenue + list growth | Owned-audience health (currently near zero — should climb fast) |
| Engagement rate + saves/shares per pillar | What content earns reach in this community |
| Group-buy: sign-ups → orders, MOQ progress, tier conversion | Whether events are pacing |
| Follower growth per platform | Top-of-funnel health |
| AOV + returning-customer rate | High-ticket repeat behavior |

**Cadence:** weekly async report (what posted, what performed, next week's plan) → monthly
review vs. the calendar → quarterly reset of channels/tools with Nate.
